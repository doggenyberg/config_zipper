With this program, you are able to copy your CS:GO config files faster. Make sure to know where your config files are stored before using this program.

  1. 1

    Import your CS:GO config folder. The program will zip the necessary files and save them to your computer.

  2. 2

    Assuming you will play on another computer, you will have to save the zipped folder manually. For example to a USB stick or cloud storage.

  3. 3

    Upload the zipped folder to your game folder on the new/other computer.
